# 🛒 Retail Sales & Customer Insights

An end-to-end Data Analytics project that explores customer shopping behavior using Python, SQL, and Power BI. This project demonstrates the complete analytics workflow, from data cleaning and exploration to SQL analysis, interactive dashboard creation, and business reporting.

---

## 📌 Project Overview

The objective of this project is to analyze customer shopping behavior and retail sales data to uncover meaningful business insights. The project includes data preprocessing, exploratory data analysis (EDA), SQL-based analysis, Power BI dashboard creation, and presentation of findings.

This project showcases practical data analytics skills used in real-world business environments.

---

## 📂 Dataset

- Customer Shopping Behavior Dataset
- Format: CSV
- Contains customer purchase records, product categories, payment methods, demographics, shopping preferences, and transaction details.

## 📈 Project Workflow

### 1️⃣ Data Collection

- Imported customer shopping dataset
- Loaded data into Python

### 2️⃣ Data Cleaning

- Checked missing values
- Removed duplicates
- Corrected data types
- Prepared clean dataset for analysis

### 3️⃣ Exploratory Data Analysis (EDA)

- Customer demographics
- Product category analysis
- Spending behavior
- Shopping trends
- Payment method analysis
- Purchase frequency

### 4️⃣ SQL Analysis

Performed SQL queries to answer business questions such as:

- Total sales
- Customer segmentation
- Category-wise performance
- Average purchase amount
- Spending trends
- Customer insights

### 5️⃣ Power BI Dashboard

Created an interactive dashboard including:

- KPI Cards
- Sales Overview
- Customer Insights
- Product Category Performance
- Payment Method Analysis
- Interactive Filters & Slicers

### 6️⃣ Business Report

Prepared a presentation summarizing:

- Key Findings
- Business Insights
- Recommendations
- Dashboard Screenshots
